Geoswift

Geoswift is an international financial service provider with 5 international office and many exchange outlet in China. The increasing demand of cross border remittance has provided a lot of business opportunities to Geoswift

Challenges

In order to facilitate efficient and seamless fund remittance, Geoswift is required to build a secure and easy to use web services for their client. Workflow process and audit trail is important to make sure the operation stays complaint.

Our Solution

- Role based administrator and merchant portal
- Easy to implement secure RESTful Web Service
- Workflow and audit trail feature to ensure it stays complaint
- PCI-DSS compliant system

Technology used

- Java
- Spring Security
- RESTful Web Services
- Microsoft SQL Server

Result

Trioloop has helped Geoswift to build a secure yet easy to use API and web-based merchant portal for their client make remittance to and from China. The system is PCI-DSS compliant which provide extra confident to the consumer and merchants. It is integrated with multiple gateways such as China Union Pay, Tenpay and so on allowing different payment and remittance options.
